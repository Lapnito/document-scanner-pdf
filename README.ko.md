## 문서 스캐너 PDF란?

**문서 스캐너 PDF**는 Android와 iPhone을 위한 군더더기 없이 본질에 집중한 모바일 스캐너입니다. 카메라를 문서에 비추면 앱이 자동으로 가장자리를 잡고, 원근을 보정하며, 페이지를 향상시키고, 멀티페이지 PDF로 내보냅니다. 모든 처리가 기기 내에서 — 계정 없음, 업로드 없음, 구독 없음.

실제로 사람들이 스캔하는 것들을 위해 만들어졌습니다: 영수증, 청구서, 계약서, 신분증, 학교 과제, 명함, 손글씨 메모, 강의 핸드아웃, 책, 이메일로 보낼 또는 포털에 업로드할 양식.

## 주요 기능

### 캡처
- **카메라 스캔** + 자동 가장자리 감지
- **사진에서 가져오기** — 기존 이미지를 스캔된 페이지로
- **멀티페이지 캡처** — 필요한 만큼 페이지 추가

### 정리
- **자동 원근 보정** — 비스듬히 찍어도 평평한 직사각형 페이지로
- **수동 가장자리 조정** — 자동이 빗나갈 때
- **문서 필터** — 깔끔한 흑백, 그레이스케일, 컬러, 매직 클린
- **밝기와 대비** 로 마무리

### 정돈
- **멀티페이지 문서** — 페이지 추가, 삭제, 드래그로 순서 변경
- **로컬 라이브러리** — 스캔은 기기에 남음

### 내보내기
- **PDF 내보내기** — 화질 옵션(소형 / 표준 / 고화질)
- **공유** — 이메일, 메신저, 클라우드 등 PDF를 받는 어떤 앱에든
- **저장** — 기기 또는 즐겨 쓰는 클라우드 폴더에

## 작동 방식

**가장자리 감지는 어떻게 되나요?**
앱은 카메라의 각 프레임을 분석하여 문서에 가장 가까운 직사각형을 찾습니다. 모서리는 수동으로 다듬을 수 있습니다.

**"매직 클린"은 무엇인가요?**
대비를 올리고 그림자를 제거하여 얼룩진 종이 사진을 깨끗한 흰색 페이지와 또렷한 텍스트로 바꿉니다.

**스캔본이 어디로 업로드되나요?**
아닙니다. 모든 처리는 기기 내에서 이뤄집니다.

**갤러리의 여러 사진을 하나의 PDF로 만들 수 있나요?**
가능합니다. 한 장씩 가져와 순서를 정한 뒤 한 PDF로 내보냅니다.

## 자주 묻는 질문

**정말 무료인가요?**
네. 유료 기능은 없습니다.

**계정이 필요한가요?**
아닙니다.

**문서가 업로드되나요?**
아닙니다.

**한 PDF에 최대 몇 페이지?**
고정 제한 없음. 기기 저장공간만이 한계.

**저장된 스캔을 수정할 수 있나요?**
네 — 다시 열어 페이지 추가/삭제, 필터 재적용, 다시 내보내기 가능.

**왜 카메라 권한이 필요한가요?**
페이지를 캡처하는 유일한 방법이기 때문입니다. 스캔 외에는 쓰이지 않습니다.

**왜 사진 권한이 필요한가요?**
기존 이미지를 고르고 PDF를 저장하기 위해서입니다.

**OCR이 되나요?**
본 버전은 깔끔하고 공유 가능한 PDF에 집중합니다. 검색 가능한 OCR은 로드맵에 있습니다.

**어떤 기기를 지원하나요?**
카메라가 있는 Android와 iOS 13.0 이상의 iPhone / iPad.

**버그 신고는 어떻게?**
[github.com/Lapnito/document-scanner-pdf/issues](https://github.com/Lapnito/document-scanner-pdf/issues)에 issue를 등록하거나 tom@lapnito.cz로 메일.
