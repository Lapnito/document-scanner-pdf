## ドキュメントスキャナー PDF とは？

**ドキュメントスキャナー PDF** は Android と iPhone のための、余計なものを省いた集中型のモバイルスキャナーです。書類にカメラを向け、エッジを自動検出させ、パースを補正、ページを整え、複数ページの PDF として書き出します。すべての処理は端末内で完結 — アカウント不要、アップロードなし、サブスクなし。

実際に人がスキャンする物のために設計：レシート、請求書、契約書、身分証、宿題、名刺、手書きノート、配布資料、本、メールで送ったりポータルに上げたりするフォーム。

## 主な機能

### 取り込み
- **カメラスキャン** とエッジ自動検出
- **写真からインポート** — 既存画像をスキャンページに
- **複数ページの取り込み** — 必要なだけページを追加

### クリーンアップ
- **自動パース補正** — 斜めから撮ったページもまっすぐで矩形に
- **手動でのエッジ調整** — 自動が外れたら微調整
- **ドキュメントフィルター** — クリーンな白黒、グレースケール、カラー、マジッククリーン
- **明るさ／コントラスト** で仕上げ

### 整理
- **複数ページ書類** — 追加、削除、ドラッグで並び替え
- **ローカルライブラリ** — スキャンは端末内に

### 書き出し
- **PDF 書き出し** に画質オプション（小 / 標準 / 高）
- **共有** メール、メッセンジャー、クラウド — PDF が使える先ならどこへでも
- **保存** 端末内、またはお気に入りのクラウドフォルダへ

## 仕組み

**どうやってエッジを見つけているの？**
カメラの各フレームを解析し、書類に最も近い矩形を探します。角は手動で微調整可能。

**「マジッククリーン」とは？**
コントラストを上げ、影を取り除き、汚れた紙の写真を白くて文字の鮮明なページに変えます。

**スキャンは送信される？**
されません。すべて端末内で処理されます。

**ギャラリーの複数枚から 1 つの PDF を作れる？**
作れます。1 枚ずつインポートし、並び替え、まとめて 1 つの PDF として書き出します。

## よくある質問

**本当に無料ですか？**
はい。有料機能はありません。

**アカウントは必要ですか？**
不要です。

**ドキュメントは送信されますか？**
されません。

**1 つの PDF の最大ページ数は？**
固定の上限なし。ストレージのみが制限。

**保存後に編集できますか？**
できます — 再オープン、ページの追加／削除、フィルター再適用、再書き出し。

**カメラ権限はなぜ必要？**
ページを撮る唯一の方法だからです。スキャン以外には使いません。

**写真権限はなぜ必要？**
既存の画像を選び、PDF を保存するためだけです。

**OCR はある？**
本バージョンは綺麗で共有しやすい PDF を作ることに集中。検索可能な OCR はロードマップ上。

**対応端末は？**
カメラ付きの Android、iOS 13.0 以降の iPhone / iPad。

**不具合の報告は？**
[github.com/Lapnito/document-scanner-pdf/issues](https://github.com/Lapnito/document-scanner-pdf/issues) に issue を、または tom@lapnito.cz までご連絡。
