## เครื่องสแกนเอกสาร PDF คืออะไร?

**เครื่องสแกนเอกสาร PDF** คือสแกนเนอร์มือถือที่เน้นเฉพาะสิ่งจำเป็น ไม่มีของฟุ่มเฟือย ใช้ได้บน Android และ iPhone จ่อกล้องไปที่เอกสาร ให้แอปตรวจหาขอบอัตโนมัติ แก้มุมมอง ปรับให้ดูดี และส่งออกเป็น PDF หลายหน้า ทุกขั้นตอนทำในเครื่อง ไม่ต้องสมัคร ไม่อัปโหลด ไม่มีการสมัครสมาชิก

ออกแบบมาสำหรับสิ่งที่คนสแกนกันจริง ๆ ใบเสร็จ ใบกำกับ สัญญา บัตรประชาชน การบ้าน นามบัตร โน้ตเขียนมือ เอกสารบรรยาย หนังสือ และฟอร์มที่ต้องส่งอีเมลหรืออัปโหลดบนพอร์ทัล

## คุณสมบัติหลัก

### การถ่าย
- **สแกนด้วยกล้อง** พร้อมตรวจหาขอบอัตโนมัติ
- **นำเข้าจากรูปถ่าย** — เปลี่ยนภาพที่มีอยู่ให้กลายเป็นหน้าที่สแกนแล้ว
- **ถ่ายหลายหน้า** — เพิ่มหน้าได้ตามต้องการ

### ทำความสะอาด
- **แก้มุมมองอัตโนมัติ** — แม้ถ่ายเฉียงก็ได้หน้ากระดาษเรียบและตั้งฉาก
- **ปรับขอบด้วยมือ** กรณีอัตโนมัติพลาด
- **ฟิลเตอร์เอกสาร** — ขาวดำคม สเกลเทา สี ทำความสะอาดเวทมนตร์
- **ความสว่างและคอนทราสต์** สำหรับปรับละเอียด

### จัดระเบียบ
- **เอกสารหลายหน้า** — เพิ่ม ลบ และลากเพื่อจัดลำดับ
- **คลังภายใน** — ภาพสแกนอยู่ในเครื่อง

### ส่งออก
- **ส่งออก PDF** พร้อมตัวเลือกคุณภาพ (เล็ก / มาตรฐาน / สูง)
- **แชร์** ทางอีเมล แชต คลาวด์ — ที่ใดก็ตามที่รองรับ PDF
- **บันทึก** ลงเครื่องหรือโฟลเดอร์คลาวด์ที่ชอบ

## กรณีใช้งาน

| สถานการณ์ | สิ่งที่แอปทำ |
|-----------|--------------|
| ใบเสร็จค่าใช้จ่าย | สแกน ตัด ทำความสะอาดเวทมนตร์ และ PDF หน้าเดียว |
| สัญญาหลายหน้า | จับภาพทั้งหมด จัดลำดับ ส่งออกเป็น PDF เดียว |
| บัตรประชาชน | ด้านหน้าและหลังในไฟล์ PDF เดียวกัน |
| รูปถ่ายกระดานไวต์บอร์ด | แก้สีให้อ่านได้ |
| โน้ตเขียนมือ | ฟิลเตอร์ขาวดำให้เส้นคม |
| เอกสารบรรยาย | คลังภายในเครื่อง |
| นามบัตร | แชร์เข้ารายชื่อผู้ติดต่อ |
| การบ้าน | ส่ง PDF ให้ครู |
| ฟอร์มประกัน | อัปโหลด PDF บนพอร์ทัล |
| อัลบั้มภาพเก่า | สำรองดิจิทัลในรูปแบบ PDF |

## ทำงานอย่างไร

**การตรวจหาขอบทำอย่างไร?**
แอปวิเคราะห์ทุกเฟรมจากกล้องและหารูปสี่เหลี่ยมที่ใกล้เคียงเอกสารมากที่สุด มุมต่าง ๆ ปรับด้วยมือได้

**"ทำความสะอาดเวทมนตร์" คืออะไร?**
เพิ่มคอนทราสต์ ลบเงา และเปลี่ยนภาพกระดาษเลอะเป็นหน้ากระดาษขาวกับตัวอักษรคม

**ภาพสแกนถูกอัปโหลดที่ไหนหรือไม่?**
ไม่ ทุกอย่างเกิดบนเครื่องของคุณ

**ทำ PDF จากภาพหลายภาพในแกลเลอรีได้ไหม?**
ได้ นำเข้าทีละภาพ จัดลำดับ และส่งออกเป็น PDF เดียว

## คำถามที่พบบ่อย

**ฟรีจริงไหม?**
ใช่ ไม่มีฟีเจอร์ที่ต้องเสียเงิน

**ต้องสมัครบัญชีไหม?**
ไม่ต้อง

**เอกสารถูกอัปโหลดไหม?**
ไม่ถูก

**จำกัดจำนวนหน้าใน PDF ไหม?**
ไม่มีลิมิตตายตัว ขึ้นกับพื้นที่จัดเก็บ

**แก้ไขสแกนที่บันทึกไว้แล้วได้ไหม?**
ได้ เปิดใหม่ เพิ่ม/ลบหน้า ใช้ฟิลเตอร์ใหม่ และส่งออกอีกครั้ง

**ทำไมต้องขอกล้อง?**
เพราะเป็นวิธีเดียวในการจับภาพหน้ากระดาษ ใช้กล้องเพื่อสแกนเท่านั้น

**ทำไมต้องขอสิทธิ์ภาพถ่าย?**
เพื่อให้เลือกภาพที่มีและบันทึก PDF

**ทำ OCR หรือเปล่า?**
รุ่นนี้เน้นทำ PDF ที่สะอาดพร้อมแชร์ OCR ค้นหาได้อยู่ในแผน

**รองรับอุปกรณ์ใดบ้าง?**
อุปกรณ์ Android ที่มีกล้อง และ iPhone / iPad ที่ใช้ iOS 13.0 ขึ้นไป

**แจ้งบั๊กอย่างไร?**
