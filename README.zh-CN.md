## 文档扫描器 PDF 是什么？

**文档扫描器 PDF** 是为 Android 与 iPhone 打造的、聚焦本质、毫无花哨的移动扫描器。把相机对准文件，让应用自动识别边缘，校正透视，增强页面，并导出多页 PDF。整个流程都在本机完成 — 无账号、无上传、无订阅。

它面向人们真正会扫描的东西：发票、回执、合同、身份证、作业、名片、手写笔记、讲义、书籍，以及需要发送邮件或上传到门户的表格。

## 主要功能

### 拍摄
- **相机扫描**，自动识别边缘
- **从相册导入** — 让现有照片变成扫描页
- **多页拍摄** — 想加多少页都可以

### 清理
- **自动透视校正** — 即便斜着拍，页面也会扁平且方正
- **手动调整边缘**，自动失败时备用
- **文档滤镜** — 干净黑白、灰度、彩色、魔法清洁
- **亮度与对比度** 微调

### 整理
- **多页文档** — 增、删、拖动排序
- **本地图书馆** — 扫描件留在设备上

### 导出
- **PDF 导出** 含质量选项（小 / 标准 / 高）
- **分享** 邮件、聊天、网盘 — 任何接受 PDF 的应用
- **保存** 到本机或你常用的云端目录

## 工作方式

**边缘识别如何完成？**
应用分析每一帧画面，找到最像文档的矩形。角点可手动微调。

**"魔法清洁"是什么？**
提升对比度、去除阴影，让脏旧纸张照片变成有清晰文字的白色页面。

**我的扫描会被上传吗？**
不会。一切都在你的设备上完成。

**能从相册的多张照片合成一个 PDF 吗？**
可以。逐张导入、排序、整体导出为一个 PDF。

## 常见问题

**真的免费吗？**
是。没有付费功能。

**需要账号吗？**
不需要。

**会上传我的文档吗？**
不会。

**单个 PDF 最多多少页？**
没有硬性上限，仅受存储限制。

**保存的扫描可以再编辑吗？**
可以 — 重新打开、增删页面、再次应用滤镜并重新导出。

**为何要相机权限？**
扫描页面的唯一方式。相机仅用于扫描。

**为何要照片权限？**
用于选择已有图像并保存 PDF。

**支持 OCR 吗？**
本版本聚焦于干净可分享的 PDF，可搜索 OCR 在路线图上。

**支持哪些设备？**
带相机的 Android 设备、iOS 13.0 或以上的 iPhone / iPad。

**如何反馈缺陷？**
在 [github.com/Lapnito/document-scanner-pdf/issues](https://github.com/Lapnito/document-scanner-pdf/issues) 提交 issue，或邮件至 tom@lapnito.cz。
